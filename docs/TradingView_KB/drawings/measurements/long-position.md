# Long Position**

#### ***

#### Application

The Long Position tool allows the user to set an entry point and assume a long position from that point. Extending above and below that price level will be two boxes: green for the profit zone and red for the loss zone. Both zones can be manually adjusted by the user to change the risk/reward ratio.

*#### Inputs

*Account Size

Defines the total account balance used for risk and position size calculations.

Lot Size

Determines the minimal amount of traded instruments.

Risk

Defines the amount of risk per trade.

Entry Price

Allows for precise placement of trade's entry point. 

Leverage

Defines the leverage applied to the position. Leverage affects margin requirements.

Profit Level

Defines the target level at which the trade will be closed with profit.

Stop Level

Defines the level at which the trade will be closed to limit losses.

Ticks

Allows you to set the profit target or stop-loss as a distance in ticks from the entry price. The system automatically converts the specified number of ticks into a profit price or stop price based on the instrument's tick size.

Price

Allows you to manually specify the exact price level at which the profit target or stop-loss should be placed.

QTY precision

Controls the rounding and display precision of the quantity value.

#### Style

*Lines

Can change the color and thickness of the line separating profit/loss areas as well as the profit and loss targets.

Stop Color

Sets the color for the stop-loss region.

Target Color

Sets the color for the profit target region.

Text

Can select the text color and font size.

Price labels

Toggles the visibility of the drawing's price values on the price axis.

Stats

Controls which data is displayed on the Risk/Reward tool.

*Compact stats mode

Shortens the position performance tags so they take up less space on the chart.

Always show stats

If this checkbox is not selected, Stats will appear beside only when the Long Position is selected by mouse click.

#### Visibility

In the Visibility properties dialog, you can toggle displaying of the Long Position on charts of different timeframes. For any timeframe you can select either to show or to hide it.
